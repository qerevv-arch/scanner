# data/ — модульные файлы «бурения»
Эти JSON-файлы подгружаются `index.html` и управляют источниками данных и правилами оценки.
Редактируй их без переписывания кода.

- providers.config.json — порядок источников и включение/выключение модулей.
- additives_ru.json — словарь пищевых добавок (E-номера) на русском.
- allergens_ru.json — словарь аллергенов.
- local_rules_il.json — локальные пороги и метки для Израиля.
- alternatives_seed.json — стартовые альтернативы по категориям.
- brands_map.json — нормализация брендов/синонимов.
- il_price_feeds_sources.json — источники прайс-фидов (URL заменить на реальные).
